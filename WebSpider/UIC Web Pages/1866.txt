<h1 align="center"> Generalized Conditional Gradient (GCG)</h1>
<h3 align="center">&nbsp;</h3>
<br>
<h2> Overview </h2>
<p> GCG is an open source Matlab&nbsp;solver for gauge (norm)
regularized problems, that are commonly used in sparse coding and
compressive sensing. &nbsp;Examples include matrix completion,
dictionary learning, and structured sparse estimation.&nbsp;</p>
The code includes the implementations used for all experiments in [1].
<h2> Download </h2>
Latest: <a href="./gcg.zip">GCG version 2.0</a>&nbsp;<br><br>
Old stuff: <a href="./boosting.zip">GCG version 1.0</a>&nbsp;<br><br>
This version 1.0 implements the boosting algorithm in [2].
Note the loss
function for matrix completion needs to be customized in order to
efficiently compute f(U V') where U and V have a small number of
columns. &nbsp;We provide a helper function <a href="./proj_omega_sub_single.cpp">here</a>.<br>
