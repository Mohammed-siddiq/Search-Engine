                    <h2 id="story" class="__title invisible">story</h2>

    <figure class="_image captioned-image image-right-third">
  <img src="https://sisl.lab.uic.edu/wp-content/uploads/sites/330/2018/09/usenix-symposium-website.png" alt="symposium website">
  </figure>
  
  <p><span style="font-size: 16px;">A research team from Professor V.N. Venkatakrishnan’s <a href="https://sisl.rites.uic.edu/" target="_blank" rel="noopener">Systems &amp; Internet Security Lab</a> (SISL) including PhD student Abeer Alhuzali, Rigel Gjomemo, PhD, and Birhanu Eshete, PhD, were honored with a <a href="https://www.usenix.org/conferences/best-papers" target="_blank" rel="noopener">Distinguished Paper award</a> at the 27th USENIX Security Symposium held in Baltimore, MD from August 15-17th. Their paper was one of five receiving the honor.</span></p>
<p><span style="font-size: 16px;">“USENIX Security is the most prestigious venue in security for practical systems work, and this  award is quite prestigious in the security community,” said Venkatakrishnan.</span></p>
<p><span style="font-size: 20px;"><strong>ABSTRACT</strong></span></p>
<p>Modern multi-tier web applications are composed of several dynamic features, which make their vulnerability analysis challenging from a purely static analysis perspective. We describe an approach that overcomes the challenges posed by the dynamic nature of web applications. Our approach combines dynamic analysis that is guided by static analysis techniques in order to automatically identify vulnerabilities and build working exploits. Our approach is implemented and evaluated in NAVEX, a tool that can scale the process of automatic vulnerability analysis and exploit generation to large applications and to multiple classes of vulnerabilities. In our experiments,<br />
we were able to use NAVEX over a codebase of 3.2 million lines of PHP code, and construct 204 exploits in the code that was analyzed.</p>
