<h1 align="center"> Bayesian Online Multilabel Classification (BOMC)</h1>
<h3 align="center"> &nbsp;</h3>
<h4 align="center"> version 1.0</h4>
<h4 align="center"> September 29, 2011</h4>



<h2> Overview </h2>
<p> 
BOMC is an open source toolkit&nbsp; for online multilabel classification using 
Bayesian models [1, 2].&nbsp; It is implemented in F# 1.9.3.4 on Microsoft 
Visual Studio 2008, and can be compiled and run on Linux systems via
<a href="http://www.mono-project.com/Main_Page">Mono</a>.&nbsp; The graphical 
model, as shown in Figure 1~3 of [1], is extend from TrueSkill<sup>TM</sup> [2] 
to deal with multilabel, and the inference engine is expectation propagation.</p>

<p>We refer the interested users to two other illustrative F# implementations of 
TrueSkill<sup>TM</sup> :
<a href="http://blogs.technet.com/b/apg/archive/2008/06/16/trueskill-in-f.aspx">
original</a> [2] and
<a href="http://blogs.technet.com/b/apg/archive/2008/04/05/trueskill-through-time.aspx">
temporal</a> [3].</p>
