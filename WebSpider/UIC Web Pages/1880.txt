<h1 align="center"> Smoothing for Multivariate Scores (SMS)</h1>
<h3 align="center"> &nbsp;</h3>
<h4 align="center"> version 2.0</h4>
<h4 align="center"> August 13, 2012</h4>



<h2> Overview </h2>
<p> 
SMS is an open source, extensible and scalable convex solver for a number of machine learning problems cast in the form of regularized risk
minimization problem [1].&nbsp; It is particularly advantageous for optimizing 
multivariate performance measure [2].&nbsp; The implementation is "extensible" because the (problem-specific)
loss function modules are encapsulated with a common interface for the main 
optimizer.&nbsp; Thus it is very simple to incorporate solutions to new 
problems.&nbsp;&nbsp; At present, SMS can solve the
following problems:

</p>
<h4>
<ol>
  <li> Binary Hinge Loss<li> Precision/Recall Break-even Point (PRBEP) [2]<li> 
	Area under Receiver Operating Characteristic curve (ROCArea) [2]</li>
</ol>
</h4>

<p>SMS is implemented with optimized efficiency.&nbsp; It features parallel 
optimization, which is fuelled by the
<a href="http://www.mcs.anl.gov/petsc/index.html">PETSc</a> and
<a href="http://www.mcs.anl.gov/research/projects/tao">TAO</a> packages.</p>
<p>We also provide <a href="../data/index.html">data files</a> in PETSc format, which can be parsed 
and loaded an order of magnitude faster than the conventional LibSVM format.</p>
