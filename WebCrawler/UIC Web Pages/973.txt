       <h1>UIC Password Management</h1>

<p>The One ID and Password (OIDPW) phase of the University of Illinois Identity and Access Management (IAM) Project (<a href="http://web.uillinois.edu/IAM" target="_blank">http://web.uillinois.edu/IAM</a>) was successfully implemented at the UIC campus on October 17, 2016. For more information about the One ID and Password project please see <a href="http://accc.uic.edu/news/2016/10/17/new-system-managing-your-university-illinois-password" target="_blank">ACCC News &amp; Alerts</a>.
<p>

<h2>New Password Manager</h2>
<ul>
<li>The new Password Manager&nbsp;is available at:&nbsp;<a href="http://identity.uillinois.edu">http://identity.uillinois.edu</a></li>
<li>Use the Password Manager to set up new password recovery options because any ACCC password recovery options you may have set up in the past will be retired.</li>
<li>Use the new University of Illinois Password Management system to manage the password for both your UIC NetID and your Enterprise ID, which will be synced upon completion of your next password reset.</li>
</ul>

<h2>Legacy Password Manager</h2>
<p>There are a few circumstances when you should use our local UIC Password Manager instead:</p>
<ul>
<li>You need to change the password on a secondary NetID instead of your primary NetID. To do so, login with your secondary netid when prompted.  If you are already logged in with your primary netid, you must close your browser and start over, using your secondary netid to login.</li>
<li>Your UIC password has expired and you have never used the new Password Manager.</li>
<li>You need to change the password on an account type that doesn't use the same password as your NetID (such as ADSM or Voicemail).</li>
<li>If any of the above apply to you, then proceed to the legacy UIC Password Manager: <a href="http://changepassword.uic.edu">http://changepassword.uic.edu</a></li>
<li>If you forgot your UIC password <strong>and</strong> you have never used the new Password Manager please visit our <a href="https://accountportal.uic.edu/token/manage?dispatch=cr_panel">Account Portal page</a>.</li>
</ul>
