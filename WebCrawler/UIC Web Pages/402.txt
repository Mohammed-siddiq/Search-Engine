    </div><h1><a id="top"></a>Get pop-up alerts for UIC websites</h1>
<p style="background-color: #efefef; padding: 10px;"><strong>UIC Web Alerts</strong> are popups that can be integrated into UIC websites and will be activated automatically during a campus emergency. Alerts will include a brief summary of the emergency and a link to the UIC homepage.</p>
<h2>How it works</h2>
<p>During a campus emergency, Public Affairs will post brief web alerts, advising the campus community about the situation. When a visitor arrives on a properly coded web page, a message box will display that states the nature of the emergency and link to the UIC homepage.</p>
<p>When the alert message is minimized, the emergency status page will open in a new browser window. Each subsequent alert will be displayed when it is posted and the information on the emergency status page will be updated.</p>
<p><a href="https://emergency.uic.edu/files/2015/06/RAVE-UICAlert-1.png" rel="attachment wp-att-288"><img class="alignnone size-full wp-image-288" src="https://emergency.uic.edu/files/2015/06/RAVE-UICAlert-1.png" alt="RAVE-UICAlert" width="576" height="420" srcset="https://emergency.uic.edu/files/2015/06/RAVE-UICAlert-1.png 576w, https://emergency.uic.edu/files/2015/06/RAVE-UICAlert-1-300x219.png 300w" sizes="(max-width: 576px) 100vw, 576px" /></a></p>
<h2>How to integrate alerts into your site</h2>
<p>To integrate UIC web alerts into your website, insert a <code>&lt;script&gt;</code> tag immediately before the closing <code>&lt;/body&gt;</code> tag on web pages or templates of your site. If there are no active web alerts, pages will function as usual. The popup will appear only when a UIC web alert is sent by Public Affairs or the UIC Police. <em>Sites with a previous implementation of UIC Alerts do not need to be changed since a redirect is in place, but we encourage webmasters to update their websites to keep current.</em></p>
<p>The tag to insert is:</p>
<div style="background-color: #efefef; padding: 20px; color: #314f7b;"><code>&lt;script type="text/javascript"<br />
src="//www.uic.edu/apps/web-alert/alert.js"&gt;<br />
&lt;/script&gt;<br />
</code></div>
<p>&nbsp;</p>
<p>The code is the same whether your site is served over http or https.</p>
<h2>How to test the alerts</h2>
<p>You can preview how the web alert will work on your website by inserting a temporary testing <code>&lt;script&gt;</code> tag. This test is functional and behaves exactly like the live version. If you see an alert, it is inserted correctly. If you minimize the alert you will need to clear your cookies to redisplay the message. Be sure to remove or comment out the test tag when you are finished testing.</p>
<p>The test script tag to insert is:</p>
<div style="background-color: #efefef; padding: 20px; color: #314f7b;"><code>&lt;script type="text/javascript"<br />
src="//www.uic.edu/apps/web-alert/test.js"&gt;<br />
&lt;/script&gt;<br />
</code></div>
<h2>How to customize the positioning</h2>
<p>To change the placement of the web alert popup box add the following CSS to your page changing the <code>top</code> and <code>left</code> values to the desired position:</p>
<div style="background-color: #efefef; padding: 20px; color: #314f7b;"><code>.uic-ewas-alert-box {<br />
top:50px;<br />
left:0px;<br />
}<br />
</code></div>
<p>To change the placement of the emergency minimized box add the following CSS to your page and change the <code>top</code> and <code>right</code> values to the desired position:</p>
<div style="background-color: #efefef; padding: 20px; color: #314f7b;"><code>.uic-ewas-mini-box {<br />
display: block;<br />
right: 0px;<br />
top: 0px;<br />
}<br />
</code></div>
<h2>Contact</h2>
<p>If you have any questions about the emergency web alerts, please contact <a href="mailto:www@uic.edu">www@uic.edu</a>.</p>
